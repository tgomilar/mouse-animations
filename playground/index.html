Mouse Animations

A lightweight, framework-agnostic TypeScript library for cursor and mouse effects: trail, ripple, custom cursor, magnetic, particles, parallax, and tilt.
Also available as a jQuery plugin.

Zero dependencies.

Under 5 kB gzipped.

jQuery Plugin

All effects are available as jQuery methods. Import the plugin once — it extends $.fn automatically.

Installation 
npm install mouse-animations jquery
Import 
import 'mouse-animations/jquery';

// extends $.fn — no return value needed
Usage 
// Global effects
$('body').trail({ color: '#a78bfa', size: 8 });
$('body').ripple({ duration: 700, maxSize: 120 });
$('body').customCursor({ innerColor: '#a78bfa', smoothness: 0.12 });
$('body').particles({ count: 24 });

// Magnetic — selector auto-derived from the jQuery set
$('.btn').magnetic({ strength: 0.4, radius: 120 });

// Parallax — elements shift on mouse move
$('.layer').parallax({ depth: 20, ease: 0.1 });

// Tilt — 3D perspective tilt with optional glare
$('.card').tilt({ maxTilt: 15, perspective: 800, glare: true });

// enable / disable / destroy on any plugin
$('body').trail('disable');
$('body').trail('enable');
$('body').trail('destroy');
